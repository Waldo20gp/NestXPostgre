# STARTS-SHOP API
1. Clonar proyecto
2. Instalar dependencias
``` 
npm install
```
3. Clonar el archivo ```.env.template``` y renombrarlo como ```.env```
4. Cambiar las variables de entorno
5. Levantar base de datos
```
docker-compose up -d
```

6. Iniciar: 
```
npm run start:dev
```
